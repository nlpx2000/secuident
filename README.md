# secuident

## 介绍

本仓库是书生*浦语internlm2-chat学习提交练习作业的的仓库。

![logo](./image/mlp.png)

## 目的
通过书生浦语大模型的实战学习，搭建一个物体检测识别，语音、文本、图像、查询、知识库结合大模型理解行为的应用。

## 模型
使用internlm2-chat_7b预训练模型微调。

## 技术
- gradio
- 语音识别
- 语音合成
- 多模态
- 物体检测
